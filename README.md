# kplocker-ui

基于element-ui（2.15.6）、vue（2.6.11）封装常用业务组件，需单独安装element-ui依赖并引入

## Install
```
npm install kplocker-ui -S
```

## Quick Start
```
// main.js中全局引入
import Vue from 'vue'
import kplockerUI from 'kplocker-ui'

Vue.use(kplockerUI)

// or
import { StDialog } from 'kplocker-ui'
Vue.use(StDialog)
```
